# Авангард - PHP лендинг автосервиса

Готовый одностраничный лендинг для PHP-хостинга. Главный сценарий сайта - звонок в автосервис.

## Файлы

- `index.php` - страница, SEO-разметка, телефоны и часы работы.
- `assets/css/styles.css` - адаптивные стили.
- `assets/js/main.js` - выбор услуги, подсказка для звонка и прокрутка к телефонам.
- `sitemap.xml.php` + `.htaccess` - динамический sitemap по текущему домену.
- `robots.txt` - базовые правила индексации.

## Запуск локально

```bash
php -S 127.0.0.1:8080
```

После запуска откройте `http://127.0.0.1:8080`.

## Перед публикацией

Проверьте актуальность телефонов, графика работы и списка услуг. Если хостинг не Apache, настройте маршрут `/sitemap.xml` на `sitemap.xml.php` вручную или используйте sitemap-файл с доменом.
